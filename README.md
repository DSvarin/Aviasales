# Учебный проект "Aviasales"
## Описание
Список билетов с фильтрацией по количеству пересадок и сортировкой самый быстрый/самый дешевый.

## Стек 
* React 
  * React Hook
* Redux
* REST API (fetch, async/await)
* SCSS, стилевые модули
* PropTypes
* date-fns, nanoid
